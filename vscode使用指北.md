vscode使用指北
==

此说明不会讲解过多的功能，主要写明下载、安装、汉化、扩展安装、萌化的内容，也会推荐个人爱用的扩展，其他细节设置不做说明。

[toc]

## 1.简要介绍与下载安装

> 微软在2015年4月30日Build 开发者大会上正式宣布了 Visual Studio Code 项目：一个运行于 Mac OS X、Windows和 Linux 之上的，针对于编写现代 Web 和云应用的跨平台源代码编辑器。

个人认为VSCode是目前最好的IDE

下载网站：[Visual Studio Code - Code Editing. Redefined](https://code.visualstudio.com/)

![image-20240428115859981](https://raw.githubusercontent.com/ruinasS/imgs/main/image-20240428115859981.png)

软件的安装没有什么特别不同的，不过安装中有一个选项推荐勾选

## 5.功能

### emmet语法支持

> 一个Web开发工具，用于加快HTML和CSS代码的编写速度。
> 使用 Emmet 能够通过简短的表达式就可以生成HTML或CSS代码片段。

文档网址：[emmet文档](https://docs.emmet.io/cheat-sheet/)
